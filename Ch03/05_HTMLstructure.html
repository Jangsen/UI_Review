HTML5 기본 태그를 사용하여 작성한 문서입니다.

태그 이름은 대소문자를 구분하지 않지만, HTML5 문서는 소문자로 작성할 것을 권장

공백처리 문제

HTML 5에서는 엔터, 탭 스페이스바와 같은 키보드 값을 인식하지 않음

HTML5에서는 스페이스바 대신 특수문자를 사용

태그의 포함관계

태그의 포함관계를 표현하기 위해 들여쓰기 적용

    잘 정리된 문서

  • 문서의 구조를 명확하게 작성
  • 들여쓰기를 해야 소스코드의 가독성이 높아짐
  • 줄 바꿈도 해야함

    잘 정리가 안된 문서

  • 문서의 구조를 명확하게 작성하기
  • 들여쓰기를 해야 소스코드의 가독성이 높아짐
  • 줄바꿈도 해야함
